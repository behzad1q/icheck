icheck

icheck is plugin to drupal form element.
This plugin provide some effect on checkboxes and radio buttons.

Steps:-
1. Download icheck library form "http://fronteed.com/iCheck/".
2. Extract it and rename to 'iCheck'.
3. Place it in libraries folder.
4. Download module.
5. Place it in modules folder.
6. Enable module.
7. Go to 'admin/config/icheck/icheck' page.
8. Configure settings.
9. Save the settings.
10. Now you can see changes in thr form's checkbox and radio buttons.
